Наблюдатель/Observer

Наблюдатель это поведенческий паттерн, который позволяет одним объектам реагировать на изменения происходящие в других объектах

Псевдокод класса

class Observer {
    private listeners = [];

    subscribe(listener) {
        listeners.add(listener);
    }
    unsubscribe(listener) {
        listeners.remove(listener);
    }
    notify(param) {
        listeners.forEach((listener) => listener(param));
    }
}

Одиночка/Singleton

Одиночка это порождающий паттерн, который обеспечит существование только одного экземпляра класса

Псевдокод класса

class Singleton {
    private static instance = new Singleton();
    private constructor() {}

    static getInstance() {
        return Singleton.instance;
    }
}
Псевдокод использования класса

const first = Singleton.getInstance();
const second = Singleton.getInstance();
first === second //true
