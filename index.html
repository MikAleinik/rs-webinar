SPA

Содержание.
  • Вкратце о SPA приложениях.
  • Реализация простого SPA.
Что такое SPA приложение.
  • Это реализация веб-приложения, которое загружает только один веб-документ, а затем обновляет содержимое тела этого единственного документа с помощью API-интерфейсов JavaScript, таких как и Fetch, XMLHttpRequest когда нужно показать другой контент. (MDN)
Признаки SPA приложения.
  • Маршрутизация между компонентами по строке запроса браузера.
  • Генерирует собственные элементы без участия сервера.
  • Хранит собственное состояние при переходах.
Особенности SPA приложения.
  • Трудности индексации страниц(ы) поисковыми системами.
  • Более сложная, при первоначальном знакомстве, реализация.
  • Требуется переадресация запросов на стороне сервера на единую точку входа.
Действия браузера при загрузке приложения.
  • Отправляет запрос на получение исходного html документа.
  • Последовательный разбор (парсинг) полученного html документа.
  • Загрузка файлов стилей.
  • Загрузка файлов скриптов.
  • Создание DOM.
  • Создание CSSOM.
  • Рендер, компоновка и отрисовка.

Выделенные пункты это то, на чем SPA может получить преимущество перед MPA. В остальных - будет зависеть от реализации приложения.
