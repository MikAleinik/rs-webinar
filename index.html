Shelter

Содержание.
  • Компонент слайдер изображений.
  • Компонент пагинации страниц.
  • Просмотр решений студентов.
Что попробуем.
  • Разделение кода на модули.
  • Создание компонентов страницы в коде.
Модули.
  • Разделяют код на отдельные части.
  • Структурируют код.
  • Увеличивают переиспользование кода.
  • Убирают дублирование кода.
Синтаксис.

//file my-library.js
function someFunction() {
    //Какой-то код...
}
export { someFunction };
                    

import { someFunction } from './my-library';
import * as MyLibrary from './my-library';

someFunction();
MyLibrary.someFunction();
Особенности.
  • Требуют правильной последовательности подключения скриптов.
  • Или использования сборщиков :) .
Создание компонентов в коде.
  • "Разбиваем" наш html на модули/элементы.
  • Создаем и добавляем наш элемент с актуальным состоянием.
  • Избегаем поиска элементов при инициализации.
  • Создаем и/или изменяем страницу на лету.
  • Первый шаг сохранению состояния приложения при переходах на другие страницы.
